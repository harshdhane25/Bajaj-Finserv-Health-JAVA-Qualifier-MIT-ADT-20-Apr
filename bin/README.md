# Ecommerse

Simple Spring Boot e-commerce API built for Postman testing.

## Default credentials

- `admin / admin123`
- `tester / test123`

## Run locally

```powershell
cd D:\codex\ecommerse
.\mvnw.cmd spring-boot:run
```

## Main endpoints

- `GET /api/public/health`
- `GET /api/products`
- `GET /api/products/{id}`
- `POST /api/products`
- `GET /api/auth/me`
- `GET /api/debug/headers`
- `POST /api/orders`
- `GET /api/orders/my`
- `GET /api/orders`
